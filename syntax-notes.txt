# New Bzo Syntax

---

Nothing is final right now. This is all just rough sketches.


Operators:
+	add
-	sub
*	mul
/	div
%	mod
^	exp
<	less
>	greater
=<	less/equal
>=	greater/equal
=	equal
!=	not equal
!	not
&	and
|	or
:=	assign
@	namespace

---

Identifiers and Literals:

id		Function/variable identifier
Id		Type/interface identifier
~id		Mutable/effect variable
#id		Builtin identifier

42		integer literal
3.14	float literal
"foo"	string literal
'bar'	tag literal

---

#: this is a comment
#{
	This is a
	multiline comment
#}

---

Function Call
[foo: x, y, z]

Function Pipeline
[foo: w, x] [bar: _, y, z]


---

Function Definition
fn :: [Int, Int] -> [Int, Int]
fn :: [x, y]{
	x + y
}
